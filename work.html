Meetup

In this project I aimed to develop a serverless progressive web application (PWA) using React and a test-driven development (TDD) approach. The application leverages the Google Calendar API to fetch upcoming events, combining the benefits of serverless architecture and PWAs. The key features include filtering events by city, showing/hiding event details, specifying the number of events, offline functionality, and charts for data visualization. The technical requirements encompass React, Google Calendar API with OAuth2 authentication, serverless functions using AWS Lambda, cross-browser compatibility, responsiveness, PWA checklist compliance, offline support, installation on home screens and an alert system using an OOP approach. Through this project, I gained in depth knowledge of serverless functions, unit testing, integration testing, user acceptance testing, continuous delivery, object-oriented programming, progressive web apps, and data visualization.
